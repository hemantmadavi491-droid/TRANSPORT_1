# 🚛 Transport Management System (TransLogix)

A professional dashboard application to manage vehicles, drivers, customers, and trips — all in one place.  
Built for logistics companies, fleet managers, and transport operators to streamline operations.

---

## ✨ Features
- **Dashboard Overview**: Quick stats on vehicles, drivers, customers, and trips.
- **Vehicle Management**: Add, edit, and track fleet details.
- **Driver Management**: Maintain driver profiles and assignments.
- **Customer Management**: Store customer information and link them to trips.
- **Trip Management**: Create and monitor trips with assigned vehicles and drivers.
- **Reports**: Generate insights for better decision-making.

---

## 🛠️ Tech Stack
- **Frontend**: React.js (with dark-themed UI)
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Authentication**: JWT-based secure login
- **UI Framework**: TailwindCSS / Bootstrap (depending on your setup)
